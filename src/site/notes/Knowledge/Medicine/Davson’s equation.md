# Davson’s equation

cf [[Darcy's law\|Darcy's law]] & [[Ohm's law\|Ohm's law]]

ICP = Rout x If + Pd

If = formation of [CSF](Cerebrospinal%20fluid.md)
Pd = Pressure of dural venous sinus

Because I(formation) = I(resorption) at steady state
